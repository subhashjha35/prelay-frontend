Schedule API

{{"http://prelay-api.herokuapp.com/v1/schedule/{service}/{departure_station_id}/{arrival_station_id}/{from_timestamp}?key=TRIAL"}}
service can be 'mav' only as for now.
departure_station_id: id of a station from the below mentioned endpoint.
arrival_station_id: id of a station from the below mentioned endpoint.
from_timestamp: utc midnight timestamp or any given timestamp from which the schedule should be returned.

Stations API

{{"http://prelay-api.herokuapp.com/v1/stations/{service}?key=TRIAL"}}
service can be 'mav' only as for now.
It's an array of stations, station properties should be defined as well.
